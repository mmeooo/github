## GitHub 공유

1. `cd 원하는 폴더`
2. `git clone 깃허브 주소`
3. 파이참에서 clone한 파일 열기


깃에서 누가 만들어놓은 파일 받기-> pull (↙)

깃으로 내가 파일 올리기-> push(↗)

깃의 최신버전과 내 파이참이 다르면 메세지 창이 뜸. merge 클릭하기


# 충돌이 일어나는 상황

1. 같은 파일에 같은 라인을 수정했을 때
2. 파일이 다를 때
